How to install

To start using PHP, you can:

  • Find a web host with php and MySql support
  • Install a web server on your own PC, and then install PHP and MySql

Many PHP developers like to build their websites and test on localhost before publishing to live server.
To build local web server environment on Windows computer,
For it you need to Install XAMPP on Windows.
Then you can test your code much more quickly because you donâ€™t need to deploy it to the live server
every time you make a change. Once all changes are completed you deploy once.

What is XAMPP?

XAMPP is a free distribution package that makes it easy to install Apache Web Server, PHP, Perl and MySQL.
It is a simple, lightweight Apache distribution, itâ€™s extremely easy to use for developers to create a
local web server for testing purposes.

Download and Install XAMPP On Windows

To download and install XAMPP on your windows use the following steps:

1. Got to Apache Friends Website to download the latest version of XAMPP for Windows.

s

2. After download, click the xampp-windows-x64-8.1-.6-0-VS16-installer.exe file to launch the installer

During the installation process, sometimes you may receive warnings such as Important notice.
Just click ok and proceed with the installation.

3. Next you will see the Welcome to the XAMPP Setup Wizard screen. Click the Next button to continue.

4. All components are checked by default so no need to change anything and click the Next button to continue.

5. Next you can choose a folder to install XAMPP, in my case itâ€™s C:/xampp as i am installing XAMPP in drive C,
you can set yours where you would like to install and click the Next button to continue.

6. Then you will be prompted to choose a language. Just choose English and click the Next button to continue.

7. A Bitnami info will show up . If you want to learn more about bitnami leave
the checkbox checked else unchecked it and click the Next button to continue..

8. After the installation is done just click Finish and a XAMPP control Panel will show up.
