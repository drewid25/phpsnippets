Get Started

1. Open the XAMPP app in your machine. Initialize the XAMPP server by pressing the start button at the right side of Apache and MySql module.

2. After initializing the XAMPP server, locate the XAMPP folder inside your machine.
In my case I installed the XAMPP inside my drive C:\. Next step is to open your command line and do the following steps.
*Make sure you are in the C:\ drive before you perform the steps. If not just type cd .. until you reach the c:\ .

  • Step 1.Inside the command line type cd XAMPP to go to XAMPP folder.
  • Step 2.Type cd htdocs to go htdocs folder.
  • Step 3.Inside htdocs create a folder. type mkdir foldername to make a folder.
  • Step 4.Type cd foldername to open the folder.
  • Step 5.Type code . to open the folder inside vscode.

3. Once the folder is open in VScode. Create a new file and name it index.php press enter.

4. After creating an index.php file initialize the php by copying the code below. .

5. Then open your browser and type localhost/foldername/index.php and you will be able to see the Hello World output on your screen as seen below.

6. Congrats ! you have started a php program.
